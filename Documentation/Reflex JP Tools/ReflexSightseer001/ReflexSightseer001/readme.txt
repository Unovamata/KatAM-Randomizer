●概要
　配布元ページに書いてあるとおりのグラフィック取り出しツールです。

●インストールとアンインストール
　インストールは適当なフォルダに入れてください。
　アンインストールはそのフォルダを消してください。

●説明
　あらかじめ、このソフトのフォルダに、「星のカービィ鏡の大迷宮」のイメージを
kagami.gba
　という名前にして置いておいてください。
　そして、ソフトを起動し、ボタンをポチポチ押せば、同じフォルダに画像が出力されるはずです。
（合計１１Ｍバイト程度です）

●その他
　出力されたものは、ＲＯＭと同様の扱いをしてください。
　数点ほど、少々、珍しいものも出力されるようです。
